** Details **

* 정의 및 특징 * 
	=> 부가적인 세부사항(additional details)을 명시할 때 사용.
	=> 보통 사용자가 직접 접거나 펼 수 있는 대화형 위젯(interactive widget)을 정의할 때 사용되며,
	   내부에는 어떠한 종류의 콘텐츠도 포함될 수 있음.
* 하위Tag : summary * 
	=> details 내부에서 화면에  표시될 제목 (visible heading)을 명시.
	=> 이 제목을 마우스로 클릭함으로써 details 요소를 보일수도, 숨길수도 있음.
* 사용예 : 오늘의 환율 *
  • 달러($) : 1,135.90
  • 유로(€) : 1,284.08
  • 엔(￥) : 1,014.88
