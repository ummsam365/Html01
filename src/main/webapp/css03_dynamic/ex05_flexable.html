** 플랙서블 레이아웃 **

=> Block 은 기본적으로 뷰에 수직으로 쌓이기 때문에 수직 구성은 쉽게 구현이 가능 하지만,
   상대적으로 수평 구성은 구현하기 쉽지 않음.
=> 과거에는 수평구현을 위해 테이블, float, inline-block 을 사용했으나,
=> display: flex; 를 사용하면 더 쉬운 구현이 가능함.
=> flex-wrap:플렉스 라인에 공간이 부족할때 플랙스 요소의 위치를 다음줄로 넘길지 여부를 결정
=> justify-content: 플렉스 요소의 수평방향 정렬방식을 설정하는 것
