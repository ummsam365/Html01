** Float **
=> 왼쪽 또는 오른쪽으로 객체를 띄워 정렬하는속성
=> 본래의 취지는 부유하는 대상을 위해 만들었지만, 실제는 Layout을 위해 많이 사용됨.
=> 웹페이지 만들때 가장 많이 사용하는 속성중의 하나.

** Clear **
=> Tag가 float 속성에 영향 받지 않도록 clear
	- clear:left -> float:left 속성 취소
	- clear:right -> float:right 속성 취소  
	- clear:both -> left, right 모두 취소
