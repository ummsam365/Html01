** Color 투명도

** Size 단위 **

1) px: pixels (a dot on the computer screen), 크기, 위치

2) pt: point (1 pt = 1/72 inch), 절대크기, 글자크기

3) pc: pica(또는 picas, 파이카) 행의 폭으로 글자크기를 나타내는 단위 -> 1pc는 12pt

4) em: 1em 은 현재 폰트의 크기와 같습니다. _______2em 은 현재 폰트 크기의 2배 크기를 의미합니다. _______만약 현재폰트(Parent Tag 기준) 의 크기가 12pt이면 2em은 24pt 크기에 해당합니다.

5) %: 비율
