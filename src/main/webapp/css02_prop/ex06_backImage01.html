	** background-attachment
	=> 페이지가 스크롤 될 경우 배경의 이미지를 고정시킬 것인지를 지정
	=> fixed 또는 scroll
	
	** background-size 
	=> 배경이미지의 사이즈를 조정
	=> auto : 기본값은 auto, 이미지의 원본 크기로 그대로 설정.
	=> cover : 이미지의 비율을 유지하고자 할때.
			(현재 엘리먼트의 가로와 세로에 이미지를 맞추면서 비율을 유지함)
	=> contain : 이미지의 비율을 유지하고자 할때.
			(부모 엘리먼트 내부에 비율이 고정된 이미지가 들어가며,
			  이미지의 가로와 세로 중 큰 값을 기준으로 비율이 고정됨.)
	=> px 또는 % : 가로와 세로의 크기를 지정하여 사용.
