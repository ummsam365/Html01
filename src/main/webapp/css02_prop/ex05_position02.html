** Position 에 따른 문제점
=> absolute 사용시 다른요소에 무관하게 무조건 위치하므로 다른 태그들이 가려짐
	-> z-index (값이 클수록 우선순위를 가짐)
=> 상위 요소를 벗어나는 부분에 대한 해결
	-> overflow

** Position과 연관되는 속성 **
=> overflow : 부모 Tag 에 주로 적용 
=> z-index : 표시되는(겹침) 순서, 
             position 값을 absolute로 지정했을때만 적용됨
